**Before using for official purposes** (invoicing, demo mailing, legal, CRM): **verify each entity** on [Texas Comptroller Franchise Tax Account Status](https://comptroller.texas.gov/taxes/franchise/account-status/search) and/or [Texas SOS SOSDirect](https://direct.sos.state.tx.us), capture the **registered agent / PIR owner and mailing address** from the official record, and replace or confirm the row below.

## Official lookup (public data) — required for Comptroller-verified addresses

- **Texas Comptroller:** https://comptroller.texas.gov/taxes/franchise/account-status/search — search by Entity Name; capture mailing address, registered agent, PIR owner/director address.
- **Texas SOS:** https://direct.sos.state.tx.us — business entity search; capture registered agent and registered office.

**To get Comptroller-verified owner/director addresses:** Run the Comptroller (and SOS if needed) search for each entity above, capture the official mailing address and PIR/registered agent name and address, then update the table and set *Address source* to **Comptroller/SOS** for that row.
